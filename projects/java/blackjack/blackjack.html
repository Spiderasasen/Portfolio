Blackjack

Ah, Java — I love and hate you at the same time. I love you because of the memories I have coding with friends, like when we built our version of the Oregon Trail. But you’re also a pain for being so damn specific: if something isn’t an Object, it complains; if you mix up a String and an Int, you’re screwed. So yeah, I love you, but you drive me crazy. This project is just a straightforward blackjack game — nothing too serious. If you know how to play blackjack, you can play this. I mainly built it to have a solid Java project in my GitHub, and I’m proud it’s there.
