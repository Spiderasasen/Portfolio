Project:
Python I did for stats class

This project started as a side experiment during my stats class. We were supposed to use SAS, but I found it frustrating — too rigid, too procedural, and it sucked all the fun out of coding. Instead of wrestling with PROC statements and syntax order, I decided to try Python, where messing up feels like part of the adventure and getting it right feels amazing. Even though I didn’t turn this in for homework, the Python version gave me the same results we eventually got with SAS. It reminded me why I love coding: the challenge, the mistakes, and the satisfaction when it finally works.
