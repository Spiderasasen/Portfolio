Python SQL Chess Database

SQL is one of the best languages for storing and manipulating data — and in my opinion, far more practical than SAS. You might wonder why this project shows up in the Python section of my portfolio. The answer is simple: I used Python to integrate with SQL, inserting and checking items in a database. With over 1,000 elements to manage, writing individual insert statements in MySQL Workbench wasn’t realistic. So, for my final project in database class, I built a Python+SQL solution to automate the process. It’s the main reason I created the “Class Work” section on my GitHub, and I’m proud of how it came together.
