Project:
Positivity with letters

Something simple, but wholesome. I built this project as a little boost during midterms — because let’s be real, we all know the stress. Coding is fun for me, but sometimes it’s nice to get a few words of encouragement too. Even if you type “hello world,” it transforms into something uplifting. Type anything, and it’ll give you kind words right alongside it. You can even put in something like “Hate” and still get love back. My hope is that this project gives you the same encouragement it gave me — a reminder that even in tough times, we all deserve a little positivity.
